Writing a console App in C#

C# is an open source, object oriented programing language. Using C# we can build web app, mobile app, games app development, and widows services etc., even we can write application that run on tiny microcontroller(tiny chip) in short C# runs everywhere.

C# is use as a backend language, back-end languages are the ones that run on the server and handle the business logic and data storage of a web application.

.NET is the framework, and C# is programing language and it is a heavily integrated component of windows, it includes the virtual machine (CLR, Core language runtime).

C# has more easy features, for example, List and Dictionaries can be accessed using brackets, and properties can be accessed using dot notation instead of calling explicitly getter and setter methods like in Java.
